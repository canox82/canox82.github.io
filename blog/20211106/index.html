Come promesso ho aggiunto la pagina Argomenti...

Come avevo detto, ho aggiunto la pagina di riepilogo degli argomenti trattati. Al momento è molto semplice, si tratta solo di un “recap” dei vari tag presenti con il link che rimanda alla pagina di ricerca per i post che li contengono. Semplice ed efficace.

In pratica creo un array tags, poi scorro il JSON dei post e per ogni post controllo se il primo tag è già presente nell’array tags tramite la funzione .includes()

let tags = [];

for (let i in posts){
    for (let t in posts[i].metadata.tags) {
        if (!tags.includes(posts[i].metadata.tags[t])) {
            tags.push(posts[i].metadata.tags[t])
        }
    }
}

se non è resente lo aggiungo con .push().

Alla fine per visualizzare l’elenco tags mi basta eseguire un ciclo con each. 

<ul class="flex flex-col my-5">
    {#each tags as tag}
    <li class="flex m-1">
        <a href="/tags/{tag}">
            <div class="flex-initial py-1 px-2 mx-1 
                 rounded-lg bg-blue-500  hover:bg-blue-700">
                <p class="text-sm  text-gray-100">{tag}</p>
            </div>
        </a>
    </li>
    {/each}
</ul>

 
Per il futuro voglio aggiungere un contatore per ogni argomento, di modo da sapere quante volte è stato trattato. Non è di difficile realizzazione.

Ho anche aggiornato la pagina “About Me”, prima era troppo sintetica, ora forse è troppo prolissa ma vabbè, alla prossima magari troviamo la giusta via di mezzo.
